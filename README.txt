Cosmo Integrated Engineering Static Website

How to use:
1. Open index.html in a browser to preview the site.
2. Replace assets/cosmo-logo.png with the final logo file if needed.
3. Replace assets/work/work-01.svg to work-11.svg with the 11 real project photos.
4. The Send Email and Get a Quote buttons use mailto:infocosmoeng@gmail.com and work without a backend.

This is a static website: HTML + CSS + JavaScript only. No server or database is required.
